# Programación Asistida por IA - Ejercicios JavaScript

Este repositorio contiene el código desarrollado durante la materia de **Programación Asistida por IA** de la Maestría.

## 📋 Descripción

En este repositorio se alojan los ejercicios de **JavaScript** desarrollados durante el curso de Programación Asistida por IA. El objetivo es documentar y compartir el aprendizaje sobre el uso de herramientas de inteligencia artificial para asistir en el desarrollo de software, aplicado a la resolución de problemas de programación en JavaScript.

## 🚀 Contenido

Este repositorio incluye:
- Ejercicios de JavaScript desarrollados con asistencia de IA
- Soluciones a problemas de programación
- Ejemplos de implementación de conceptos de JavaScript
- Documentación de procesos y metodologías aprendidas

## 🛠️ Tecnologías

- **JavaScript** (ES6+)
- Node.js (para ejecución en entorno de servidor)

Las tecnologías adicionales se irán documentando en cada ejercicio específico.

## 📝 Estructura

La estructura del repositorio se organizará por proyectos o módulos del curso a medida que se vaya desarrollando el contenido.
